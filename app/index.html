Playbook

Jump start your interface development.

Currently v1.0.4

Playbook is a Yeoman generator to get you building interfaces faster. Jekyll is used for static site generation. Bourbon, Neat & Bitters is included to help you write CSS faster.

Installation

    npm install -g generator-playbook

Usage

    grunt serve
